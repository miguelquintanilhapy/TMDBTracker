# 🎬 TMDBTracker
API em desenvolvimento com Node.js + TypeScript focada em aprendizado de backend e boas práticas.

⚠️ Status: projeto ainda não utilizável

------------

## Sobre
Este projeto foi criado com o objetivo de estudar a construção de APIs utilizando Node.js e TypeScript, aplicando conceitos de organização em camadas e separação de responsabilidades.
A aplicação consome dados da API do TMDB e implementa lógica própria, como sistema de recomendação.

----------

## Estrutura do projeto
O projeto segue uma organização baseada em camadas:

* controllers → responsáveis pelas requisições e respostas
* services → regras de negócio e lógica da aplicação
* middlewares → tratamento de erros e autenticação
* config → configurações (env, banco de dados)
* routes → definição das rotas da API
* types → tipagens com TypeScript

----------
